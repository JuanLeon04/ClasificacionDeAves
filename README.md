# Clasificación automática de aves europeas por su canto mediante aprendizaje automático

<img width="800" height="300" alt="Clasificación de aves por su canto" src="https://github.com/user-attachments/assets/2cc4e706-cf52-4f10-b51d-7e1c8b2d2b19" />

Proyecto de IA para clasificación de aves a partir de cantos. Dataset con ~14k audios de 32 especies (desbalanceado), características acústicas extraídas (MFCC, contraste espectral). Se aplican algoritmos supervisados (DT, GNB, RF, SVM, DNN) y no supervisados (KMeans, DBSCAN, PCA, t-SNE) para análisis y modelado.
